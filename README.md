This repository contains all data discussed in the manuscript titled "Lattice Thermal Conductivity and Strain Response of Platinum Dichalcogenide Monolayers: A First-Principles Investigation," by Nathaniel Bunker, Stuart Shepard, and Manuel Smeu. Benchmarking data, procedures, and supplementary calculations can be accessed by reasonable request made to the Corresponding Author, Dr. Manuel Smeu: msmeu@binghamton.edu
