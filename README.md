This repository contains all data discussed explicitly in the manuscript titled "Isotope, Boundary, and Quasi-Harmonic Strain Effects on Lattice Thermal Conductivity for Platinum Dichalcogenide Monolayers: A First-Principles Investigation," by Nathaniel Bunker, Stuart Shepard, and Manuel Smeu. Benchmarking data, procedures, and supplementary calculations can be accessed by reasonable request made to the Corresponding Author, Dr. Manuel Smeu: msmeu@binghamton.edu
